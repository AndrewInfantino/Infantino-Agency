# Infantino Branding & Marketing — Website

## Deployment (GitHub + Vercel)

### Folder Structure
```
your-repo/
├── index.html
└── images/
    ├── Infantino-Logo.svg         ← Your logo SVG
    ├── Andrew-Enhanced.png        ← Hero photo (arms crossed)
    ├── Andrew-Infantino_Photo.png ← About section photo (seated)
    ├── image6.png                 ← Food bg: ravioli (restaurant industry card)
    ├── image7.png                 ← Food bg: bowl (real estate industry card)
    ├── image8.png                 ← Food bg: burger (online biz industry card)
    ├── image37.png                ← Copper Branch logo
    ├── image45.svg                ← Rio's Pizza logo
    ├── image53.svg                ← Organito Foods logo
    ├── image55.svg                ← The Poke Guy logo
    ├── image57.svg                ← MIG Audit logo
    ├── image59.svg                ← Skybird Asian Grill logo
    ├── image65.svg                ← OMJ Fitness / Le M Nord logo
    ├── image66.png                ← GoldX² logo
    ├── image67.png                ← Real Food Love logo
    └── image68.png                ← Don't Eat the Cake! logo
```

### Steps
1. Create a new GitHub repository
2. Add `index.html` to the root
3. Create an `images/` folder and add all image files (rename them exactly as above)
4. Push to GitHub
5. Go to vercel.com → New Project → Import your GitHub repo
6. Vercel auto-detects static HTML — click Deploy
7. Connect your GoDaddy domain in Vercel's domain settings
